Caesar Cipher Cryptography

2024 Open Source Encryption

Overview

Caesar Cipher adalah algoritma enkripsi sederhana yang menggeser setiap huruf dalam teks dengan jumlah posisi yang sama di alfabet. Proyek ini mengimplementasikan enkripsi dan dekripsi Caesar Cipher dengan antarmuka yang user-friendly.

Key Features

  • Enkripsi & Dekripsi Text
  • Shift Cipher dengan Custom Shift Value
  • Brute Force Attack Demo
  • Analisis Frekuensi Karakter

Technologies Used

Python JavaScript HTML5 CSS3
